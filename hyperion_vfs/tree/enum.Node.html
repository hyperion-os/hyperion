Enum hyperion_vfs::tree::Node

source · 
pub enum Node<Mut> {
    File(FileRef<Mut>),
    Directory(DirRef<Mut>),
}

Variants§

§

File(FileRef<Mut>)

a normal file, like /etc/fstab

or

a device mapped to a file, like /dev/fb0

§

Directory(DirRef<Mut>)

a directory with 0 or more files, like /home/

or

a device mapped to a directory, like /https/archlinux/org/

mountpoints are also directory devices

directory devices may be unlistable, because it’s not sensible to list every website there is

a directory device most likely contains more directory devices, like /https/archlinux/org inside /https/archlinux/
